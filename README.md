# 问卷星球项目

## 项目说明

本项目是一个实现了问卷星基本功能的 React 项目，能够实现问卷的设计、填写、统计等功能

## 本地运行方式

1. 安装 node.js
2. clone 本仓库到本地
3. cd 到仓库文件夹
4. 运行命令 npm install
5. 运行命令 npm run start

完成上述步骤后，项目将运行在本地的 3000 端口

## 常用脚本

1. npm run start，用于在本地运行项目
2. npm run build，用于将项目编译打包
